1.Закинь файлы на веб сервер в папку домена.
2.Отредактируйте config.php в корне, заполните там все поля.
Поля там __TG_TOKEN это телеграмм токен бота.
BOT_USERNAME = это никнейм бота в телеграмме.
BOT_MESSAGE_PHOTO_URL = это ссылка на фотку для сообщения в телеграмме
BOT_CHAT_ID_NOTIFY = айди чата в телеграмме в который бот будет отправлять уведомления о новых заходах.
DOWNLOAD_URL_EXE = Ссылка на файл.
PROGRAM_NAME_EXE = Название файла с его расширением, т.е конец ссылки на файл.
3.Привяжите вебхук для бота через
https://api.telegram.org/botВАШ_ТОКЕН_БОТА/setWebhook?url=ссылка_до_файла_бот.php
4.Всё работает, можно ещё посмотреть chat_id после установки вебхука бота, написать /id в группу и бот напишет chat_id.
